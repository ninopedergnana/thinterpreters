This Website can parse LOOP, WHILE and GOTO programs

A simple editor which parses your LOOP and WHILE programs to prettified Haskell and runs them with the given input parameters. Make sure to check the documentation if you are stuck or try copying and pasting some examples.

Enter Code

Return value is given by x0.

Enter Parameters

Parameters (x1, x2, x3, ..., xn) must be natural numbers separated by Comma, i.e. "2, 15"
